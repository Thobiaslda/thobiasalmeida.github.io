📊 Keeping Data Current

This model is built to evolve. All team data lives in the const TEAMS block inside this page's <script> tag. To update before a specific match, find the team key (e.g. BRA) and edit: preparation.score for injury/controversy status, form.wins/draws/losses for recent results, or notes for context. Save and refresh.

Key fields to watch: preparation.score (0–100, drops with injuries/controversy), tactics.flexibility (rises when squad has versatile depth), squad.experienceScore (adjust for first-time WC squads).

Data is estimated and updated periodically — last reviewed June 2026.
